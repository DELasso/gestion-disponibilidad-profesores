# Sistema de Gestión de Disponibilidad de Profesores

Este repositorio es un **monorepo** que contiene dos aplicaciones:

- `frontend/` - Aplicación web desarrollada en **React 18**, sirve como interfaz para estudiantes y docentes.
- `backend/` - (en desarrollo) API para la gestión de horarios y usuarios.

---

## 📦 Tecnologías utilizadas

| Parte     | Tecnología         | Versión  |
|-----------|--------------------|----------|
| Frontend  | React              | ^18.x    |
| Frontend  | Node.js            | ^20.x    |
| Producción | Nginx              | alpine   |
| Contenedores | Docker & Docker Compose | latest   |
